## NDA Warning

Xbox TCR requirements are covered by NDA. The `tcr.md` file in this directory
is a stub. Fill it from the Microsoft Game Dev portal after signing the
developer agreement. **Do not commit actual TCR content to a public repository.**

## Dev Access Requirements

- Microsoft Game Developer account (ID@Xbox for indie developers)
- Xbox devkit hardware (or PC in devkit mode for early dev)
- Microsoft GDK installed
- Submission via Partner Center

## Key Submission Milestones

| Milestone | Typical Lead Time |
|-----------|------------------|
| First submission | 5–10 business days |
| Resubmission | 3–5 business days |
| Expedited review | Contact ID@Xbox rep |

## Platform-Specific Notes

- GDK targets Xbox One, Xbox Series X/S, and PC via the same SDK
- Xbox Play Anywhere: single purchase covers Xbox + PC (optional program)
- Smart Delivery: automatic delivery of best version per hardware (requires implementation)
- Xbox Game Pass: separate agreement with Microsoft if desired
- Achievements are mandatory (Xbox Achievement system)
