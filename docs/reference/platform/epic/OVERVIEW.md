## Knowledge Gap Warning

EGS policies, EOS SDK versions, and store requirements have evolved significantly.
Verify current requirements via the developer portal before submission.

## Dev Access Requirements

- Epic Games developer account
- Application submitted to EGS (Epic reviews all submissions — not open like Steam)
- EOS SDK integrated (mandatory for EGS titles)
- Build submitted via Epic dev portal

## Key Submission Milestones

| Milestone | Typical Lead Time |
|-----------|------------------|
| Initial application approval | 2–4 weeks |
| Build + store page review | 1–2 weeks |
| Resubmission | ~1 week |

## Platform-Specific Notes

- EGS is curated — not all submissions are accepted
- EOS SDK integration is required: minimum features include Auth, Achievements
- Epic First Run program: exclusivity window in exchange for better revenue share (optional)
- Achievements are supported and expected
- Cloud saves: supported via EOS, not required but recommended
- No formal cert checklist like console — Epic reviews at their discretion
- Cross-play via EOS is a selling point Epic promotes

## Revenue Share

- Standard: 88% developer / 12% Epic (as of last verification)
- Epic First Run (exclusivity): better split — check current terms at dev portal
