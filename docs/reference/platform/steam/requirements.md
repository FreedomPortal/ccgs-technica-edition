# Steam — Store Page & Asset Requirements

Last verified: 2026-06-06  
Official source: https://partner.steamgames.com/doc/store/assets

## Capsule Images (Required)

| Asset | Dimensions | Format | Notes |
|-------|-----------|--------|-------|
| Header Capsule | 460×215 px | JPG/PNG | Main browse image |
| Small Capsule | 231×87 px | JPG/PNG | Search results |
| Main Capsule | 616×353 px | JPG/PNG | Featured placement |
| Vertical Capsule | 374×448 px | JPG/PNG | Mobile/new UI |
| Page Background | 1438×810 px | JPG/PNG | Optional, darkened |
| Bundle Header | 707×232 px | JPG/PNG | If selling in bundle |

## Screenshots

- Minimum: 5 screenshots
- Dimensions: 1280×720 or 1920×1080 (16:9)
- Format: JPG or PNG
- Must show actual gameplay (no marketing text, no UI mockups)
- Valve may reject if screenshots show content not in the game

## Trailer

- Recommended: at least one trailer
- Upload via YouTube link OR direct upload in partner portal
- First trailer shown on store page = "primary" trailer
- Autoplay trailer: first 30 seconds matter most

## Store Text

| Field | Limit | Notes |
|-------|-------|-------|
| Short Description | 300 chars | Appears in search/browse |
| Long Description | No hard limit | Supports basic HTML |
| About This Game | Separate from long desc | First thing players read |

## System Requirements

- Must specify minimum AND recommended for each platform
- Fields: OS, Processor, Memory, Graphics, DirectX, Storage, Sound Card (optional)
- Verify minimum specs actually run the game before submitting

## Content Descriptors

Fill in all applicable mature content tags (violence, language, sexual content).
Inaccurate descriptors are a ToS violation and can result in removal.

## Age Rating

IARC questionnaire in partner portal — automated, no cost, same-day.
Generates ratings for: PEGI, ESRB, USK, ACB, CERO (not official for CERO), ClassInd.
