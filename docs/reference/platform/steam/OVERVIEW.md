## Knowledge Gap Warning

Steamworks SDK versions and store policies update frequently. Verify SDK version
and API changes at the partner portal before each release.

## Dev Access Requirements

- Steam Partner account (Steamworks)
- App ID issued by Valve after Direct fee ($100 USD per app, as of last verification)
- Build uploaded via SteamPipe (steamcmd or Steamworks SDK tools)

## Key Submission Milestones

| Milestone | Typical Lead Time |
|-----------|------------------|
| App ID creation | Immediately after Direct fee |
| Store page review | 3–5 business days |
| Build review (first release) | 3–5 business days |
| Age rating (via IARC) | Same-day (automated) |

## Platform-Specific Notes

- No formal cert process — Valve reviews store pages and builds, not TRC-style checklists
- IARC age rating is integrated into the partner portal (questionnaire-based)
- Steam Deck compatibility rating is separate and optional but recommended
- Early Access is a formal program, not just a label
