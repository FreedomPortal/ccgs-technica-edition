## Overview

itch.io is an open marketplace with no review or certification process. Upload
and publish immediately. No revenue share minimum — you set your own split
(default: 10% to itch.io, configurable down to 0%).

## Dev Access Requirements

- itch.io account (free)
- No application or approval needed
- Upload via web dashboard or butler CLI tool

## Key Facts

| Property | Value |
|----------|-------|
| Revenue share | You choose (0–30% to itch.io, default 10%) |
| File size limit | No hard limit (contact itch for very large files) |
| Platforms supported | Windows, Mac, Linux, Android (APK), HTML5, any file |
| Pricing models | Paid, PWYW (pay-what-you-want), free, free with optional donation |
| Age rating | Self-declared (SFW / Mature content toggle) |

## Platform-Specific Notes

- HTML5 games run directly in browser (itch iframe embed)
- itch.io is popular for game jams — `butler` CLI integrates with jam workflows
- No DRM system — all downloads are DRM-free by default
- Community features: devlogs, comments, community forums per project
- itch.io Collections and Featured placement exist but are not a formal program
- Revenue via Stripe or PayPal; regional availability varies
